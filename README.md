# Study Tracker 📚

A personal productivity tool designed to track study sessions, analyze accuracy, and maintain a consistent learning habit. Built for students preparing for competitive exams.

## Features
- **Log Sessions:** Record your subject, chapter, questions attempted, and correct answers.
- **Persistent Data:** Uses JSON to save your progress locally.
- **Visual Statistics:** Get insights into your subject-wise accuracy using interactive charts.
- **Web Interface:** Built with Streamlit for a seamless, responsive experience.

## Live Demo
Check out the live application here: https://study-tracker-m29rmpeijsxyiatnrre5rh.streamlit.app

## Technologies Used
- **Language:** Python
- **Frontend/UI:** Streamlit
- **Data Storage:** JSON

## Why I built this
As a student preparing for board exams, I needed a data-driven way to track my progress and identify which subjects require more focus. This project helped me bridge the gap between simple terminal scripting and building web applications.

## How to run locally
1. Clone the repository: `git clone https://github.com/your-username/study-tracker.git`
2. Install dependencies: `pip install streamlit`
3. Run the app: `streamlit run study_tracker_streamlit.py`
